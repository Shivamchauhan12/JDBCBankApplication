# GitHub Repository Documentation

## Project Overview
This GitHub repository contains a Java application that manages user accounts and performs various operations related to user accounts. The project is organized into several Java files, each of which serves a specific purpose. Below, you'll find a brief description of each file in the repository.

### Java Files
1. **CheckUserBalance.java**: This file handles the functionality to check the balance of a user's account.

2. **CheckUserDetails.java**: Contains code for checking and displaying user account details.

3. **CreateAccount.java**: Manages the creation of new user accounts.

4. **CreditAmount.java**: Implements the logic for crediting a specified amount to a user's account.

5. **DebitAmount.java**: Contains code for debiting a specified amount from a user's account.

6. **InputTaker.java**: Manages user input and data retrieval for various operations.

7. **JdbcConnManager.java**: Handles the database connection and data access through JDBC.

8. **Main.java**: The main class that serves as the entry point for the application. It orchestrates the different operations and their execution.

9. **UpdateUserDetails.java**: Contains functionality to update user account details.

10. **UserExceptionHandling.java**: Manages exception handling for the application.

### Getting Started
To run and use this application, follow these steps:

1. Clone the repository to your local machine using `git clone`.

2. Compile the Java files using your preferred Java development environment or the command line.

3. Make sure you have a compatible database system set up and configure the connection details in `JdbcConnManager.java`.

4. Execute the `Main.java` file to start the application.

### Usage
The application provides the following features:
- Create a new user account.
- Check user account balance.
- Check user account details.
- Credit a specified amount to a user's account.
- Debit a specified amount from a user's account.
- Update user account details.
